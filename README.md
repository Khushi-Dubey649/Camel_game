# 🐪 Camel Game

A simple and fun Camel Game built using HTML, CSS, and JavaScript. The entire project is contained in a single HTML file, making it lightweight and easy to run in any modern web browser.

## 🎮 Features

* Interactive gameplay
* Smooth animations
* Responsive design
* Built with pure HTML, CSS, and JavaScript
* No external libraries or dependencies

## 🚀 Getting Started

To run the game:

1. Download or clone the repository.
2. Open the `index.html` file in your web browser.
3. Enjoy playing the game!

## 📁 Project Structure

```text
Camel_game/
│
├── index.html
└── README.md
```

> Note: The `index.html` file contains the HTML structure, CSS styling, and JavaScript logic for the game.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🎯 How to Play

1. Open the game in your browser.
2. Use the game controls to play.
3. Avoid obstacles and try to achieve the highest score.
4. Refresh the page to restart the game.
